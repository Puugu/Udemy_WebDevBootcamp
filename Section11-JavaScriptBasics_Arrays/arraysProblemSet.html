Array Problem Set

printReverse()

This function takes an array as an argument and prints its elements in the reverse order.

isUniform()

This function takes an array as an argument and returns true if all the elements in the array are identical.

sumArray()

This function takes an array of numbers as an argument and returns the sum of all the numbers in the array.Note: It is acceptable to assume the array is full of numbers for this exercise

max()

This function accepts an array of numbers and returns the largest value in the array. Note: It is acceptable to assume the array is full of numbers for this exercise
